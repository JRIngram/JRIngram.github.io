About Me

As this was a web based project, my work was heavily focused on web technologies whilst using agile methodologies.

Featured Projects

Word_Guesser: A game written in Java, inspired by Fallout's hacking minigame. This desktop application provides the user with 10 words (either real words, or a randomly generated string of letters), and one of these words is chosen by the program as the correct word. The user has 4 guesses to find the correct word; with each guess the user is given the Hamming Distance of the chosen word vs the 'correct' word.

FiLo: A dummy website to demonstrate my knowledge of client-side and server side programming, using PHP, JavaScript, HTML and CSS. The idea behind this website is that a user would post lost items that they have found e.g. a lost necklace, another user would log-on to the website and see their lost necklace and 'claim' the item.

Genetic Algorithm Experiment: Software written originally written in Java to teach myself the basics of genetic algorithms. Currently it uses basic reproduction, crossover and mutation algorithms. In future I would like to make these more complex, and allow the user to switch between the different algorithms. In future I would like to allow this piece of open source software to become an educational tool so that others can benefit from my experimentations when learning about genetic algorithms.
